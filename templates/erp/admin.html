⚙️ How does PWA Tracking work?

1. Installation Callback: When a user accepts the PWA prompt and installs the application, the mobile browser fires a standard appinstalled event on the client side. We intercept this event and send a secure telemetry callback to /api/track-install to increment our metrics.

2. Standalone Launch Detection: When the user opens the application via the home screen shortcut, the browser launches it in standalone mode (hiding browser controls). The client-side JavaScript detects this mode via CSS media query matching and sends a callback to /api/track-launch.

3. Lightweight & Secure: All telemetry runs asynchronously in the background and does not collect personal HOD identifiers, ensuring total privacy.
